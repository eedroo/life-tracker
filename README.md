# ⏱️ Life Tracker
Aplicação web para monitorar tempo de trabalho e calcular ganhos em tempo real.

## 🚀 Funcionalidades
- Contador de tempo em tempo real
- Cálculo automático de ganhos
- Histórico de sessões
- Armazenamento local (localStorage)
- Reset de histórico

## 💻 Tecnologias
- HTML
- CSS
- JavaScript
